# فلو منصة سنتر (بعد تبسيط الهوية)

> محدّث مع الإصدار 1.3 — المدرس هو بوابة الطلاب والدفع؛ الأدمن يشرف ولا يوافق على كل حساب.

---

## القاعدة الذهبية

الطالب يشاهد درسًا أو يدخل امتحانًا فقط إذا:
1. مرتبط بالمدرس (أضافه المدرس أو وافق على طلب انضمام)
2. لديه اشتراك **نشط** على المادة مع هذا المدرس

---

## فلو الزائر (بدون تسجيل)

```
/ → صفحة سنتر
  → /teachers (كتالوج المدرسين الظاهرين)
    → /teachers/{slug} (نبذة، مواد، أسعار، فودافون كاش)
      → سجّل كطالب وانضم / أو دخول
```

---

## فلو المدرس (المسار الأساسي في السنتر)

```
1. تسجيل كمدرس (نشط فورًا) + بيانات البروفايل / فودافون / مواد
2. تفعيل «أظهر صفحتي للعامة» من البروفايل أو الريجيستر
3. من المكتب:
   - يضيف طلاب السنتر يدويًا → نشطين ومنضمين فورًا + student_code
   - يقبل طلبات انضمام من الكتالوج
   - يربط ولي أمر بطالب من مجموعته
   - ينشئ خطط اشتراك
   - نهاية الشهر: يسجّل كاش يدويًا → اشتراك active فورًا
   - أو يراجع إثبات فودافون كاش من طالب/ولي أمر → تأكيد
4. ينشر دروس وامتحانات
```

**الدفع نهاية الشهر:** الكاش يُسجَّل وقت التحصيل في السنتر بواسطة المدرس؛ فودافون كاش من **ولي الأمر** للمدرس ثم مراجعة المدرس.

**اشتراك المنصة:** المدرس يحصل على فترة مجانية (~3 شهور) ثم يدفع فودافون كاش لإدارة سنتر من `/teacher/platform`.


---

## فلو الطالب

```
تصفّح مدرس → تسجيل (نشط فورًا) → طلب انضمام
  أو: المدرس يضيفه من المكتب
→ اشتراك → دفع (كاش عند المدرس / فودافون)
→ دروس + امتحانات + شهادات
→ يشارك student_code مع ولي الأمر
```

---

## فلو ولي الأمر

```
تسجيل (نشط فورًا)
→ ربط بكود الطالب (نشط فورًا) أو المدرس يربط من مكتبه
→ متابعة KPIs
→ دفع فودافون كاش نيابة عن الابن → المدرس يراجع
```

---

## فلو الأدمن

```
هيكل أكاديمي (مراحل/صفوف/مواد)
تقارير ومدفوعات شاملة
إيقاف حساب / إخفاء مدرس من الكتالوج
ربط ولي أمر عند الحاجة
```

**لا** موافقة يومية على تسجيل طالب أو مدرس أو ولي أمر.

---

## فلو المدرس — المادة والفيديو

```
مادة واحدة فقط لكل مدرس:
  ريجيستر أو بروفايل → اختيار من الكتالوج
                   أو كتابة اسم المادة + الصف

درس فيديو (صفحة الدروس):
  1) رفع ملف → Bunny
  2) تسجيل كاميرا/شاشة من المنصة → Bunny
  3) أو لصق Bunny Video ID
```


| الدور | ماذا يفعل يوميًا |
|--------|-------------------|
| مدرس | إضافة طلاب، كاش نهاية الشهر، مراجعة فودافون، محتوى، امتحانات |
| طالب | انضمام، اشتراك، تعلم، امتحان |
| ولي أمر | ربط، متابعة، دفع فودافون للابن |
| أدمن | هيكل، تقارير، إشراف/إيقاف |
| زائر | تصفّح `/teachers` بدون حساب |
